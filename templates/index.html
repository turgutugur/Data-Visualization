Project Description

The objective of this project is to build interactive web visualizations in the form of dashboard powered by a dataset. Our dashboard focus is on NBA Statistics, specifically the data on individual teams, team comparisons, and individual players.

Our dashboard allows the user an interactive way to look up and explore:

  • 3 NBA seasons (2019, 2020, and 2021) team standings
  • All head-to-head matchups for the 2020-2021 season
  • Top player stats: points, assists, field goal accuracy, and games played for seasons 1985-2021
